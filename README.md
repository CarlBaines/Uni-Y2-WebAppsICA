# Apex Events Management System

## Project Description
This project is for my university in-course assessment. It is an ASP.NET Core Razor Pages web application which functions as an event management system. It will allow staff to manage events, guests, staff, catering and venues.

<h3>The System will use...</h3>
- A <b>Razor Pages</b> web app for the UI. <br>
- A <b>Web API (Apex.Catering)</b> for managing food, menus and event catering. <br>
- An existing <b>Apex.Venues</b> web service for venue bookings.

## Tech Stack/Technologies
- ASP.NET Core 8.0 <br>
- Entity Framework Core <br>
- Razor Pages <br>
- Web API <br>
- SQLite3 db
